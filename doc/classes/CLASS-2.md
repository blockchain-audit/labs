* [gpg](https://www.digitalocean.com/community/tutorials/how-to-use-gpg-to-encrypt-and-sign-messages)
```
# generate key
gpg --gen-key

# sign and encrypt
gpg --encrypt --sign --armor -r person@email.com name_of_file

# import public key
gpg --import name_of_pub_key_file

# decrypt
gpg file_name.asc

```
